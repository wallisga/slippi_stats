How to Use Slippi Stats Client

Installation Complete

Thank you for downloading the Slippi Stats Client! Follow the instructions below to set up and use the client.

Installation and Setup

1
Run the Installer

Locate the downloaded MSI installer file (SlippiStatsClient.msi) and double-click to run it. Follow the installation wizard to complete the setup.

You may need to allow the application through your firewall when prompted.

2
First Launch

After installation, the Slippi Stats Client will automatically launch and appear in your system tray. Look for the Slippi Stats icon in the system tray.

On first launch, you'll be asked to configure your settings.

3
Configure Settings

Right-click on the system tray icon and select Open Settings to configure the client:

  • Replay Directory: Set the path to your Slippi replay files (typically Documents/Slippi)
  • Server URL: The server is pre-configured, but you can change it if needed
  • Upload Interval: How often the client checks for and uploads new replay files

Using the Client

4
System Tray Options

Right-clicking on the system tray icon gives you these options:

  • Open Settings: Configure client settings and check connection status
  • View Logs: View upload history and any errors
  • Exit: Close the client
5
Check Upload Status

In the Settings panel, you can see the connection status and how many files have been uploaded. A green "Connected" indicator shows that the client is working properly.

You can manually trigger an upload by clicking the "Upload Now" button in the settings panel.

6
Automatic Operation

Once configured, the client will automatically run in the background and upload your replay files according to your settings. No further action is required!

The client will start automatically when you log in to your computer.

Troubleshooting

No Files Uploading
  • Verify your replay directory is correct in settings
  • Check your internet connection
  • View logs for specific error messages
  • Try clicking "Upload Now" to force an upload
Connection Issues
  • Ensure the server URL is correct
  • Check your firewall settings
  • Restart the client by right-clicking the icon and selecting "Exit", then relaunch
Need Help?

If you're experiencing issues that aren't resolved by these steps, please contact us at support@slippistats.com.
